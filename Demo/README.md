# CondenScreen Pipeline Instructions

## Overview
The CondenScreen pipeline is designed to analyze demo data efficiently and effectively, providing comprehensive insights.

## Prerequisites
- Ensure you have [Python](https://www.python.org/downloads/) installed (version 3.6 or higher).
- Install the required packages by executing:
  ```bash
  pip install -r requirements.txt
  ```

## Running the Pipeline
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/SchliekerLab/Condenscreen-v1.1.git
   cd Condenscreen-v1.1
   ```

2. **Set Up Demo Data**:
   - Download the demo data (link to be provided).
   - Place the demo data files in the `demo_data` folder.

3. **Execute the Pipeline**:
   Run the following command:
   ```bash
   python run_pipeline.py --data demo_data
   ```

4. **View Results**:
   - Results will be saved in the `results` folder.
   - Use the visualization tools provided to inspect the output.

## Troubleshooting
- If you encounter any issues, check the log files for error messages.
- Ensure all prerequisites are correctly set up.
